The <abbr> tag is used when you write an abbreviation or acronym, such as HTML or CIA. The [title] attribute specifies the long description of the abbreviation or acronym and it's shown to the user by default on mouse hover

The <code> tag is used to represent and display computer code in the markup. For example, you can represent console.log('Hello World!'). This tag gets default user agent styles that sets the font to monospace.

The <kbd> tag is used to represent keybord input the user should be prompter to type. For example: Press Ctrl + S to save.

The <samp> tag is used to represent sample output to be displayed to the user. For example:

I was trying to boot my computer, but I got this hilarious message:

Keyboard not found
Press F1 to continue

The <pre> tag is used to represent pre-formatted text. The text inside this tag will be displayed in the page exactly as written in the HTML document. For example:

The <var> tag is used to represent the name of a variable in a mathematical expression or a programming context. It's typically presented using an italicized version of the current typeface, although that behavior is browser-dependent. For example:

The volume of a box is l × w × h, where l represents the length, w the width and h the height of the box.
